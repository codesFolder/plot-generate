## How to Run

Since Insight is a client-side application, it can be hosted on any static file server.

### Local Development
1.  Navigate to the project directory:
    ```bash
    cd insight---no-code-data-analysis
    ```
2.  Serve the files using any static server. Examples:
    *   **Node.js**: `npx serve .`
    *   **Python**: `python -m http.server 3000`
    *   **VS Code**: Right-click `index.html` -> "Open with Live Server".

3.  Open your browser to `http://localhost:3000` (or the port shown clearly in your terminal).
